## GPC: Graph Prompt Clustering

## Installation

The basic packages required for the runtime environment are as follows:
```
pytorch==2.5.1
numpy==1.26.4
torch-geometric==2.5.3
scikit-learn==1.6.1
matplotlib==3.9.4
tqdm==4.66.4
```

## Run a model

python 3.9
```
sh run.sh
```

## Frozen

You can set the parameter 'is_p' to False and then save the model.
